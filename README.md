# IA Managers Toolkit

> **Tout ce qu'un manager doit savoir sur l'IA, sans le jargon technique**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![French](https://img.shields.io/badge/Lang-Français-blue)]()

## Pourquoi ce toolkit ?

Vous êtes manager et votre direction parle d'IA partout. Votre équipe tech vous présente des "modèles avec 95% de précision". Mais concrètement :

- Comment savoir si un projet IA vaut le coup ?
- Quelles questions poser pour ne pas se faire enfumer ?
- Comment piloter sans être data scientist ?

**Ce toolkit vous donne les clés.**

## Pour qui ?

- Managers non-techniques supervisant des projets IA
- Directeurs métier évaluant des solutions IA
- Product owners travaillant avec des équipes data
- Tout décideur qui veut comprendre l'IA sans devenir expert
