# Guide de Contribution

Merci de votre intérêt pour contribuer à **AI Evaluation Pratique** !

## Comment Contribuer

### Partager une Évaluation

Vous avez évalué une IA ? Votre retour d'expérience est précieux !

1. **Fork** ce repo
2. Créez un fichier dans `examples/` suivant le format existant
3. Anonymisez les données sensibles
4. Soumettez une **Pull Request**

### Améliorer le Contenu

- **Corrections** : Fautes, liens cassés, informations obsolètes
- **Clarifications** : Rendre le contenu plus accessible
- **Traductions** : Adapter en d'autres langues

### Proposer de Nouveaux Contenus

- Nouveaux templates pratiques
- Études de cas supplémentaires
- Guides sur des sujets manquants

## Standards de Qualité

Tout contenu doit être :
- **Business-oriented** : Pas de jargon technique non expliqué
- **Actionnable** : Utilisable immédiatement
- **Concret** : Avec exemples réels

## Code de Conduite

- Respect mutuel
- Critiques constructives
- Focus sur la valeur pour les managers

## Questions ?

Ouvrez une [issue](https://github.com/lafabriq/ai-evaluation-pratique/issues) ou contactez-nous à contact@lafabriq.ai
