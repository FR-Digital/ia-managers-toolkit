## Guide d'Analyse des Résultats

### Calcul du Score de Satisfaction

```
Score moyen = (Q4 + Q5 + Q6 + Q7 + Q8 + Q9) / 6
```

| Score | Interprétation |
|-------|----------------|
| > 4.0 | Excellent |
| 3.5 - 4.0 | Bon |
| 3.0 - 3.5 | Acceptable |
| < 3.0 | Problématique |

### Indicateurs Clés

**Taux de recommandation (Q9) :**
- < 3 = Détracteurs
- 3-4 = Passifs
- > 4 = Promoteurs

**Dépendance (Q15) :**
- "Très problématique" = Forte adoption
- "Un soulagement" = Rejet

### Actions Prioritaires

Basé sur les frustrations (Q11), concentrez-vous sur :
1. Les 3 frustrations les plus citées
2. Les erreurs graves (Q19-Q20)
3. Les demandes récurrentes (Q16-Q17)
